How to Use Text-to-Chart

To create a chart:

  1. Enter the values separated by commas in the text area.
  2. Check the "Include names" checkbox if you want to provide labels for each value.
  3. Select the desired chart type (Bar Chart, Line Chart, or Pie Chart).
  4. Click the "Generate" button to generate the chart.
